# Python Project

이 프로젝트는 Python으로 작성된 기본 프로젝트입니다.

## 설치 방법

1. 프로젝트를 클론합니다
2. 가상환경을 생성합니다:
   ```
   python -m venv venv
   ```
3. 가상환경을 활성화합니다:
   - Windows:
     ```
     .\venv\Scripts\activate
     ```
4. 필요한 패키지를 설치합니다:
   ```
   pip install -r requirements.txt
   ```
